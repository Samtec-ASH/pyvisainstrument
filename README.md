# PyVisaInstrument

![PyPI](https://img.shields.io/pypi/v/pyvisainstrument)

Provides boilerplate for various NI-VISA instruments and utilizes pyvisa.

- KeysightVNA
- KeysightDAQ
- KeysightPSU
- NumatoRelay (TCP/TELNET and USB)

## Installation

pip install pyvisainstrument
